### Obtener los numeros que son primos en python

```python
    def es_primo(n):
        if n < 2:
            return False
        for i in range(2, n):
            if n % i == 0:
                return False
        return True

for numero in range(1, 11):
    if es_primo(numero):
        print(numero, "es primo")
```
